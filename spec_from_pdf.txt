CAHIER DES CHARGES
Premium Fashion E-Commerce Website (COD - Algeria - Mobile First)
1. Project Overview
Objective: Develop a premium mobile-first e-commerce website for a clothing brand selling Pants,
T-Shirts, and Shoes.

Modern and premium brand identity (black & white design)

Optimized for smartphones (primary usage via Instagram traffic)

Cash on Delivery payment method

Shipping prices based on Algerian wilayas

Available in Arabic (RTL), French, and English

Integrated with Meta Pixel and Conversions API
2. Technical Stack

Next.js 14+ (App Router) with TypeScript

TailwindCSS (mobile-first styling)

PostgreSQL database with Prisma ORM

Zod validation

next-intl for multilingual support

Vercel-ready deployment

Meta Pixel + Conversions API integration
